Working pipeline

1) Import data.

Source data should have column with speaker identifiers (e.g., usernames) and one withe text (e.g., social media posts).Note: THe data for this running example are AI generated

library(speakerlandscapes)

data = read.csv("raw_corpus.csv")
head(data)
NA

2) Make data ready to build landscapes.

This create a object: a character vector in which the first token is the speaker name, prefixed by an identifying token (e.g. “agent”). Bigrams and trigrams are found in the text and replaced by single tokens (e.g., word1_word2)


ready_data = prepare_data(data, 
                          speaker_id_column = "speaker", 
                          text_column = "text", 
                          speaker_identifier = "agent")

4) Make the speaker landscape.

This step use UMAP to reduce the 250-dimension embedding to a more interpretable 2-dimension and extract speakers vectors. Other tokens present in the embedding can be also specified here to be plotted in the next step. In this exemple we use the function to extract the words that appear at least 10 times in the text.

frequent_words = get_most_frequent_tokens(ready_data, min_count = 10)
